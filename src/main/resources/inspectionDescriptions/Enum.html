Enum定义规范检查：

反例1：存在EnumOption注解，类型是class而不是enum，并且没有实现EnumFieldInterface接口 


    @EnumOption(application = "0220", description = "采购计划状态枚举")
    public class CgPlanStateEnum {
        ...
    }

反例2：存在EnumOption注解并且实现EnumFieldInterface接口，类型是class而不是enum 


    @EnumOption(application = "0220", description = "采购计划状态枚举")
    public class CgPlanStateEnum implements EnumFieldInterface {
        ...
    }

反例3：类型是enum并且实现EnumFieldInterface接口，缺少@EnumOption注解


    public enum CgPlanStateEnum implements EnumFieldInterface {
        ...
    }

枚举的正确定义方式：


    @EnumOption(application = "0220", description = "采购计划状态枚举")
    public enum CgPlanStateEnum implements EnumFieldInterface {
        ...
    }
