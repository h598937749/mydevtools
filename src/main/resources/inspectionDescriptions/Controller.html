* 针对继承自 Controller 的子类检查：

  • 是否存在Tag注解，示例： @Tag(name = "供应商应用服务")
  • 是否存在PubService注解 示例：@PubService(value = "/Budgets", prefix = RequestPrefix.API, businessCode = "02200301")
  • 是否继承Controller
  • public方法是否存在@PubAction注解，@PubAction(value = "/getBudgetsByProject", method = RequestMethod.POST)
  • 参数不能带有buguid、oid关键字
  • businessCode + value全局唯一检查
  • PubAction.value当前controller唯一检查
  • RequestBody、复杂参数检查（不能同时存在2个DTO）
