* 针对继承自 BaseEntity 的子类检查：

  • 缺少TableName注解，正例： @TableName("myUser")；级别：错误；支持快速修复
  • TableName注解缺少value属性，反例： @TableName("") 或 @TableName；级别：错误；无快速修复
  • 缺少TableId注解，正例：@TableId(value = "UserGuid", type = IdType.INPUT)；级别：错误；无快速修复
  • 当前类TableId注解数量 > 1（包含继承）；级别：错误；无快速修复
  • 实体类是抽象类（abstract）；级别：错误；快速修复：支持快速修复
